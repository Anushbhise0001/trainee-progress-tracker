# HTML Projects

Please complete the following projects and check off each one when finished.

- [ ] **Project 1: Personal Portfolio**
  - Create a personal portfolio webpage that includes:
    - A homepage with a brief introduction and overview of skills.
    - A projects section showcasing at least three projects with descriptions and links.
    - A contact form with fields for name, email, and message.
    - Use CSS to style the portfolio and make it visually appealing.
  - Submit your HTML file and a link to the live portfolio.

- [ ] **Project 2: Business Landing Page**
  - Design a landing page for a fictional business that includes:
    - A hero section with a call-to-action (CTA) button.
    - Sections for services, testimonials, and a pricing table.
    - A footer with contact information and social media links.
  - Submit your HTML file and a link to the live landing page.

- [ ] **Project 3: Responsive Blog Layout**
  - Create a responsive blog layout that includes:
    - A header with navigation links.
    - A main content area with blog posts that have titles, excerpts, and images.
    - A sidebar with links to recent posts or categories.
    - Ensure the layout adjusts for different screen sizes using media queries.
  - Submit your HTML file and a link to the live blog.

- [ ] **Project 4: E-commerce Product Page**
  - Build a product page for an e-commerce site that includes:
    - A product image gallery.
    - Product details such as name, price, and description.
    - An "Add to Cart" button and quantity selector.
    - Customer reviews and ratings section.
  - Submit your HTML file and a link to the live product page.

- [ ] **Project 5: Interactive HTML5 Quiz**
  - Develop an interactive quiz page using HTML5:
    - Create a series of questions with multiple-choice answers.
    - Include a form to submit answers and display the results.
    - Add feedback for correct and incorrect answers.
  - Submit your HTML file and a link to the live quiz.

- [ ] **Project 6: Restaurant Menu**
  - Create a webpage for a restaurant menu that includes:
    - A menu with categories (e.g., appetizers, main courses, desserts).
    - Descriptions and prices for each item.
    - An attractive design with images of menu items.
  - Submit your HTML file and a link to the live menu page.

- [ ] **Project 7: HTML5 Game**
  - Design a simple HTML5 game (e.g., tic-tac-toe or a trivia game):
    - Use HTML5 elements to structure the game.
    - Incorporate CSS for styling and JavaScript for functionality.
    - Ensure the game is interactive and user-friendly.
  - Submit your HTML file and a link to the live game.

**Submission Instructions:**
